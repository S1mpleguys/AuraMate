# 🏆 项目申报书核心内容：《“AI 气韵小师”——基于多模态大模型的居家国术康养伴侣》

## 一、 项目背景与痛点分析
1. **锻炼盲区**：八段锦、太极等中医功法强调“意、气、形”合一。居家练习缺乏专业指导，动作不到位（如关节过度伸展、重心不稳）不仅无益，反而容易造成半月板、腰椎损伤。
2. **环境忽视**：中医养生极度讲究环境，“避风如避箭”。人在运动时耗氧量大，若在密闭、空气浑浊（VOC超标）的环境中练习，会导致胸闷头晕，适得其反。
3. **交互门槛**：市面上的运动App多基于手机屏幕，老年人操作困难，且无法在运动中途进行自然地询问与交流；普通的健身镜价格昂贵，难以普及。

## 二、 硬件系统架构与外设详细作用
本系统以 **ESP-SensairShuttle 套件（ESP32-C5 为主控）** 为核心，设计为一个**可佩戴在腰带或胸前**的“智能玉佩”或“智能搭扣”。

### 1. 核心主控与板载资源
* **ESP32-C5 模组**：
  * **作用**：系统的“大脑”。负责汇聚所有传感器数据；利用其 **Wi-Fi 6** 的超低延迟特性，确保与云端大模型（如豆包大模型）进行语音对话时几乎无卡顿；利用 **BLE（低功耗蓝牙）** 接收可能扩展的外部传感器数据。

### 2. Bosch 传感器子板（套件自带）
* **BMI270 (高精度6轴IMU) & BMM350 (地磁传感器)**：
  * **作用：躯干姿态与动作连贯性评估**。
  * **具体用法**：佩戴在腰部/胸前时，它们能精准捕捉用户躯干的前倾、后仰、旋转角度（如八段锦“调理脾胃须单举”时的躯干拉伸度）。通过分析加速度和角速度的平滑度，评估用户动作是否“柔和缓慢”（太极的核心要素），而不是猛烈抽动。
* **BME690 (四合一环境传感器)**：
  * **作用：中医“环境风水”感知与呼吸监测**。
  * **具体用法**：
    * **空气质量(VOC)**：练功前检测室内挥发性有机物和污浊度，若空气差，大模型会主动语音提示：“当前房间空气略显憋闷，建议开窗通风三分钟后再练功哦。”
    * **温湿度与气压**：结合大模型判断天气，如“今天气压较低，湿度大，属于中医的‘湿气重’，建议多练两遍‘两手攀足固肾腰’”。

### 3. 人机交互外设（套件自带）
* **麦克风 & 扬声器 (I2S接口)**：
  * **作用：无障碍的双向自然语音交互**。
  * **具体用法**：摒弃复杂的按键。用户只需说“小师，我今天有点落枕，适合练什么？”麦克风采集音频，通过ESP32-C5传至云端LLM；LLM生成回复后，通过扬声器用温和的“老中医”或“小师父”音色播放。运动中途也可通过语音播报纠正动作（如：“爷爷，您的腰挺得不够直哦”）。
* **1.83英寸 LCD 屏幕**：
  * **作用：核心信息可视化**。
  * **具体用法**：显示当前的“环境健康指数”、动作达标分数（0-100分）、倒计时。UI 可设计成一个动态的“水墨风太极图”或一个可爱的“小和尚/老中医”虚拟形象。
* **RGB 灯带 (利用 GPIO/PWM 控制)**：
  * **作用：吐纳（呼吸）节拍器**。
  * **具体用法**：中医功法讲究呼吸配合动作。RGB灯带可设置为“呼吸灯”模式——灯光缓慢变亮（蓝色）引导“吸气”，缓慢变暗（绿色）引导“呼气”，帮助用户进入“心流”状态。

### 4. 扩展外设建议（加分项，体现团队动手能力）
* **扩展方案：低成本四肢动作捕捉节点 (通过 I2C 扩展或 BLE 连接)**
  * **推荐购买**：2-4个极为廉价的 MPU6050 模块（几块钱一个）+ 纽扣电池/小MCU。
  * **作用**：做成绑在手腕和脚踝上的魔术贴绑带。配合主机（腰部），形成一个低成本的全身动作捕捉网络，彻底解决单点传感器无法评判手脚协调性的问题。

## 三、 软件与算法核心逻辑（技术亮点）

为了在答辩中拿高分，你们需要展示一定的算法深度和云端架构设计：

1. **动作评估算法（DTW 动态时间规整）**：
   * 传统方法很难对比动作，因为每个人做八段锦的速度不一样。
   * **技术点**：采集专业教练的标准传感器数据作为“标准模板”。用户的实时传感器数据通过 **DTW算法**（可运行在云端服务器或边缘侧轻量级实现）与模板进行对比，计算相似度得出分数。即使老爷爷做得慢，只要轨迹对，也能得高分。
2. **大模型 Prompt 提示词工程 (RAG技术)**：
   * **技术点**：接入火山引擎大模型，但不使用默认设定。在后台构建一个“中医养生与运动康复专家”的人设。并利用 RAG（检索增强生成）技术，挂载《八段锦标准动作指南》、《黄帝内经》等文档作为知识库。
   * **效果**：让大模型的回答极其专业，避免“胡说八道（幻觉）”。
3. **“端-边-云”协同**：
   * **端侧（ESP32-C5）**：负责传感器数据采集、本地唤醒词识别、呼吸灯控制。
   * **边/云侧**：接收 ESP32-C5 通过 MQTT/HTTP 上传的音频流和 IMU 特征数据，运行复杂的动作对比算法和 LLM 推理。

## 四、 典型用户使用场景（Storytelling 讲故事）

**【场景模拟：李大爷的清晨】**
1. **环境检测启动**：早晨，李大爷拿起挂在墙上的“AI 气韵小师”戴在腰上。系统唤醒，BME690 检测到室内封闭了一夜，VOC 偏高。设备主动发声：“早上好！当前室内空气比较浑浊，为了您的肺部健康，请先开窗通风哦。”
2. **问诊与定制**：李大爷开窗后说：“小师，我昨晚没睡好，感觉肩膀有点僵硬。”大模型接收到语音，结合云端知识库回复：“好的，那我们今天重点练习八段锦中的‘左右开弓似射雕’来舒缓肩颈。请看屏幕，跟着灯光节奏调整呼吸。”
3. **纠错与陪伴**：李大爷开始练习。练到第二式时，腰部的 BMI270 和手腕的扩展传感器检测到他手臂抬得太高、重心不稳。设备立刻温柔提示：“大爷，手臂稍微放低一点，马步扎稳，重心往下沉~对，就是这样，太棒啦！”
4. **结算与关怀**：练习结束，屏幕显示今日得分 92 分。设备将训练报告推送到李大爷子女的微信小程序上，子女可以点赞互动。

## 五、 项目优势（答辩对标评审标准）

1. **直击痛点，社会价值大**：切中“人口老龄化”及“传统中医文化复兴”，相比做烂了的智能家居，具有更高立意。
2. **充分榨干套件性能**：
   * 用到了 **BME690** 的空气质量检测功能（很多队伍只会拿它测温湿度，大材小用）。
   * 用到了 **BMI/BMM** 的姿态解算。
   * 完美结合 **Wi-Fi 6** 与 **音频外设** 实现了 LLM 语音流实时交互。
3. **软硬结合的极高壁垒**：既有底层硬件驱动（I2C/I2S/SPI），又有边缘算法（姿态解算/DTW），还有云端 AI 大模型开发（Prompt/RAG），技术维度非常丰满，极受评委青睐。
